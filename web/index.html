üìã Quick Start Guide

Step 1: Configure Redis Cloud

Copy the environment template and add your Redis Cloud details:

cp .env.template .env

Edit .env with your Redis Cloud connection details from your dashboard.

Step 2: Start the Environment

docker-compose -f docker-compose-cloud.yml up -d

This will start PostgreSQL, Redis Insight, RDI CLI, and load generator containers.

Step 3: Configure RDI Pipeline

docker exec -it rdi-cli redis-di configure
docker exec -it rdi-cli redis-di deploy --config /config/config-cloud.yaml
docker exec -it rdi-cli redis-di start

Step 4: Generate Test Data

docker exec -it rdi-loadgen python /scripts/generate_load.py

This will start generating data changes in PostgreSQL that RDI will sync to Redis.

Lab 3: Test Change Data Capture

docker exec -it rdi-postgres psql -U postgres -d chinook -c "INSERT INTO \"Track\" (\"TrackId\", \"Name\", \"AlbumId\", \"MediaTypeId\", \"GenreId\", \"UnitPrice\") VALUES (9999, 'Test Track', 1, 1, 1, 0.99);"
